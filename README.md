## 💼 Projeto Destaque

### 🏥 Sistema de Painéis Hospitalares

Plataforma completa de monitoramento operacional em tempo real desenvolvida do zero para o **Hospital Anchieta Ceilândia**, com **33+ dashboards especializados** exibidos em TVs e monitores por todo o hospital, sistema de notificações multicanal e Machine Learning para previsão de demanda.

#### 📊 Arquitetura do Sistema

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   Oracle Tasy   │────▶│   Apache Hop     │────▶│   PostgreSQL    │
│   (HIS Source)  │     │   (ETL ~10min)   │     │ (DW + Histórico)│
└─────────────────┘     └──────────────────┘     └────────┬────────┘
                                                          │
       ┌──────────────────────────────┬──────────────────┼──────────────────┐
       │                              │                  │                  │
       ▼                              ▼                  ▼                  ▼
┌──────────────┐              ┌──────────────┐    ┌──────────────┐  ┌──────────────┐
│    Nginx     │              │ Notificador  │    │   Metabase   │  │  Uptime Kuma │
│ Reverse Proxy│              │     ntfy     │    │     (BI)     │  │ (Healthcheck)│
└──────┬───────┘              │ + Notificador│    └──────────────┘  └──────────────┘
       │                      │ Pareceres    │
       ▼                      │ (Apprise)    │
┌──────────────┐              └──────┬───────┘
│ Flask :5000  │                     │
│ 33 Blueprints│              ┌──────┴───────┐
└──────┬───────┘              ▼              ▼
       │                  ntfy.sh      SMTP (Email)
       │
   ┌───┴───┬─────────┬─────────┐
   ▼       ▼         ▼         ▼
Desktop  Mobile   TV/PWA   Dashboard
Browser   App     Display  Mobile
```

#### 🔧 Funcionalidades Técnicas

- ⏱️ **Atualização em tempo real** — Refresh automático configurável (60s padrão)
- 📺 **Auto-scroll inteligente** — Watchdog para displays em TV (0.5px/50ms, pausa em rodapé)
- 🔐 **RBAC granular** — Controle de permissões por painel e usuário, sessões com bcrypt
- 🔔 **Notificações multicanal** — Push (ntfy) e email (Apprise SMTP) com roteamento por especialidade médica e RTF parser para motivo de consulta
- 📱 **PWA** — Instalável em dispositivos móveis com shortcuts
- 🤖 **Análise IA clínica** — Score de risco automatizado a cada 60s por paciente (Groq/Llama 3.3 70B)
- 🧠 **Machine Learning** — Registry de modelos com métricas (MAE/MAPE/RMSE), drift detection e segmentação por área
- 📈 **ETL automatizado** — Apache Hop com ciclos ~10min, padrão TRUNCATE+INSERT e INSERT incremental para séries temporais
- 📊 **BI integrado** — Metabase conectado ao data warehouse
- 🩺 **Health checks** — Endpoints monitorados via Uptime Kuma
- 🛡️ **Segurança** — 100% das credenciais via `.env` (com validação na inicialização), bcrypt, sanitização contra SQL injection e path traversal
